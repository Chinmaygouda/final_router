# ==================== DATABASE SCHEMA ====================

PostgreSQL Tables:
1. user_conversations (semantic vault)
   ├─ id, user_id, prompt, response, model_used, tokens_consumed
   ├─ actual_cost, embedding (768-dim), created_at

2. system_logs (event tracking)
   ├─ id, user_id, event, timestamp

3. models (router registry)
   ├─ id, model_id, provider, category, tier, sub_tier
   ├─ complexity_min/max, cost_per_1m_tokens, is_active

4. conversation_archives (session history)
   ├─ id, session_id, topic_summary, full_transcript, archived_at

Redis Keys:
1. chat:{user_id} → list of messages (active session)
2. summary:{user_id} → string (session summary)
3. count:{user_id} → integer (message counter)

# ==================== TROUBLESHOOTING ====================

ISSUE: "Could not connect to Docker container (Redis)"
FIX: Either start Redis or system will work without it (PostgreSQL only)

ISSUE: "DATABASE_URL not set"
FIX: Add to .env: DATABASE_URL=postgresql://user:pass@localhost/semantic_vault

ISSUE: "APIError: Invalid API key"
FIX: Check your API keys in .env file

ISSUE: "No module named 'dispatcher'"
FIX: The imports should now work - they've been fixed

ISSUE: Vault not caching
FIX: Make sure pgvector extension is enabled: CREATE EXTENSION IF NOT EXISTS vector;

# ==================== TECHNICAL NOTES ====================

POSTGRES SETUP:
CREATE DATABASE semantic_vault;
CREATE EXTENSION IF NOT EXISTS vector;
psql semantic_vault < schema.sql (or use SQLAlchemy auto-create)

REDIS SETUP:
docker run -d -p 6379:6379 redis:latest
OR
redis-server (if installed locally)

DEPENDENCIES:
pip install fastapi uvicorn sqlalchemy psycopg2-binary pgvector python-dotenv
pip install openai anthropic google-generativeai cohere
pip install redis langchain langchain-huggingface sentence-transformers
